# The Cozy Cup Coffee Shop Website


## Project Description
The Cozy Cup is a fictional coffee shop website built 
using HTML and CSS. The website contains 5 pages and 
is designed to be fully responsive across desktop, 
tablet and mobile devices.

---

## Pages Included
- index.html - Home page
- services.html - Services and menu page
- about.html - About us page
- contact.html - Contact information page
- location.html - Location page

---

## Technologies Used
- HTML5
- CSS3 (External Stylesheet - mystyle.css)
- Responsive Design using Media Queries
- Flexbox for layout structure

---

## Features
- External CSS stylesheet linked to all pages
- Responsive navigation menu
- Mobile-friendly layout using media queries
- Hover, focus and active pseudo-classes
- Typography styling with consistent fonts
- Hero section with background image
- Two-column desktop layout using Flexbox
- Colour scheme consistent throughout

---

## How To Run
1. Open the project folder
2. Open index.html in any web browser
3. Navigate using the navigation menu
